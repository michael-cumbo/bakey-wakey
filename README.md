### Build Instruction
- Install WSL on windows.
- To install nixos use the multi user installation
https://nixos.org/download/
- Make sure to clone this repo in your home directory, aliased commands
  won't work otherwise.
- copy the config folders into the .config directory ```cp -r
  bakey-wakey/config/* .config ```
- Close and re-open WSL use these command: alias for nix-rebuild: `nh`
- Enters into the fish while also opening zellij: `fish`
